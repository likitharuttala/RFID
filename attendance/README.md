# Attendance Management System

A simple attendance management system using RFID, deployed on AWS.

## Features
- RFID-based attendance tracking
- SQLite database for storage
- Web interface for viewing records

## Prerequisites
- Python 3.8
- Flask
- SQLite
- AWS Account
